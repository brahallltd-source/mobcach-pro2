# MobCash Pro

A modern full-stack MVP built with Next.js App Router, TypeScript, Tailwind CSS and JSON persistence.

## Included
- Player / Agent / Admin dashboards
- Wallet + ledger
- Admin top-up + 10% bonus
- Order lifecycle
- Complaints + disputes
- Agent application review
- Invite bonus claim
- Notifications feed
- Modern glass UI

## Start
```bash
npm install
npm run dev
```

Open `http://localhost:3000`
